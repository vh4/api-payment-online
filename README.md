## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup CI/CD PIPELINE

Requirement =>

- nodejs v18.17.0
- npx 9.6.7

Running =>

1. npm install
2. npm run test
3. npm run build
4. start:prod
5. /api/healthy
